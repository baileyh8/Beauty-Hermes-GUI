# Beauty Hermes GUI Page Map

## Primary Surfaces

1. Chat Workbench
   - Sidebar: account, search, pinned sessions, projects, recent sessions
   - Main: thread header, messages, artifacts, composer
   - Workbench: activity, files, terminal, approval state

2. Projects
   - Project list
   - Environment status
   - Recent sessions per project
   - Archive/delete/more actions on hover

3. Settings
   - General
   - Models
   - Permissions
   - Integrations
   - Appearance
   - Advanced

## Design Constraints

- Chinese first, with product and code names left in English.
- Human messages align right; agent messages align left.
- Sidebar order is pinned, projects, recent.
- Right workbench can collapse to a single expand button.
- Composer uses a large Codex-like rounded container.
- Attachment entry is a plus button with file, folder, image, paste image, URL, and snippet actions.
